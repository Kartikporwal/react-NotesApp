# 📝 Notes App (React + Tailwind)

A simple and interactive **Notes Application** built using **React.js and Tailwind CSS**.
Users can easily add and delete notes with a clean sticky-note style interface.

---

## 🚀 Features

* ➕ Add new notes with a title and details
* 🗑 Delete notes instantly
* 🎨 Sticky notes UI design
* 📱 Responsive layout (works on mobile & desktop)
* ⚡ Fast and lightweight React app

---

## 🛠 Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **State Management:** React Hooks (`useState`)

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/notes-app.git
```

2️⃣ Navigate to the project folder

```bash
cd notes-app
```

3️⃣ Install dependencies

```bash
npm install
```

4️⃣ Start the development server

```bash
npm run dev
```

The app will run on:

```
http://localhost:5173
```

---

## 💡 How It Works

* Users enter a **note title** and **details**
* When the **Add Notes** button is clicked:

  * The note is stored in a **React state array**
  * It is rendered as a **sticky note card**
* Clicking **Delete** removes the note from the list
